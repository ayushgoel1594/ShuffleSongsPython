# ShuffleSongs
Script for shuffling the songs.

Given:
  •	Play list has 100 songs
Expectation:
  •	100 songs will be played in random manner.
  •	Same song should not be played twice.
  •	Previous and next song if we click logic should be able to handle that. 
  •	Once 100 songs are played it should stop.


Contains implementation using both JAVA and Python.
JAVA implementation is latest and more optimized.

You can find the implementation using Jave here:
https://github.com/ayushgoel1594/ShuffleSongs



Running: 
You need python compiler to run the code.

PreRequisite:
Put both the files (songsFile.txt and SongsShuffle.py) in the same folder.

The code can be used to play the current song, next song or previous song.
Code will exit when all the songs are played.
